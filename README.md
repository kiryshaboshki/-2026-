Анализ выгорания студентов

Предсказание уровня выгорания (Low/Medium/High) на основе данных об использовании ИИ.

Данные: 50 000 студентов, 16 признаков
Модель: Gaussian Naive Bayes
Инструменты: Python, Jupyter, Power BI, Git

Структура:
- data/raw/ — исходные данные
- data/processed/ — данные для Power BI
- data/results/ — графики
- notebooks/ — Jupyter ноутбуки
- powerbi/ — дашборд
- reports/ — отчет
