Анализ выгорания студентов

Проект по машинному обучению для предсказания уровня выгорания студентов на основе их привычек использования генеративного ИИ и учебной деятельности.

О проекте

Задача: Классификация уровня выгорания студентов (Low, Medium, High) на основе данных об использовании ИИ.

Данные: 50 000 студентов, 16 признаков.

Результат: Модель Gaussian Naive Bayes достигла точности 48.67%, что на 15.1% лучше случайного угадывания (baseline = 42.29%).

Результаты модели

| Метрика | Значение |
|---------|----------|
| Accuracy | 48.67% |
| Baseline (случайное угадывание) | 42.29% |
| Улучшение | +15.1% |

Отчёт по классам

| Класс | Precision | Recall | F1-score | Правильно |
|-------|-----------|--------|----------|-----------|
| High | 0.66 | 0.41 | 0.50 | 40.8% |
| Low | 0.45 | 0.73 | 0.55 | 72.5% |
| Medium | 0.47 | 0.35 | 0.40 | 34.9% |

Вывод: Модель лучше всего распознаёт студентов с низким риском выгорания (Low). Средний уровень (Medium) распознаётся хуже всего, так как он является переходным и пересекается с обоими крайними классами.

Дашборд Power BI

Интерактивный дашборд с фильтрами по специальности, курсу и политике вуза.

Скриншот дашборда

<img width="2318" height="1299" alt="image" src="https://github.com/user-attachments/assets/ba047ab2-4094-4394-a6fd-8026fe70dc50" />


Графики на дашборде:
1. Распределение уровней выгорания
2. Выгорание по специальностям 
3. Средние часы с ИИ по уровням выгорания 
4. Динамика выгорания по годам обучения 

Ключевые выводы

1. Чем больше часов с ИИ — тем выше риск выгорания. У студентов с High-выгоранием средние часы работы с ИИ значительно выше.
2. Тревожность — один из главных факторов выгорания.
3. STEM-студенты чаще других находятся в группе риска.
4. Модель лучше всего распознаёт Low-класс (72.5%), хуже всего — Medium (34.9%), так как он является переходным.
